font

説明

fontタグは明朝体、ゴシック体を指定したい句や文に使用します。

property
  • タグ名称
  • font
  • 属性
  • face
  • size
  • color
  • bold

attribute-value

face size color bold
gfont XXX% 000000 yes
mfont no

クラス変化

face size color bold
mfont font-XXXper color-000000 bold(yes)
gfont なし(no)

CSSプロパティ

mfont gfont font-XXXper color-000000 bold
font-family: serif系 font-family: sans-serif系 font-size: XXX% color: 000000 font-weight: bold
example

Code

<p><font face="mfont" size="130%" bold="yes" color="ff0000">私が明朝体です。</font></p>
<p><font face="gfont" size="130%" bold="no" color="ff0000">私がゴシック体です。</font></p>
