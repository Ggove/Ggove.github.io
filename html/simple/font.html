●font

tag name

font

attribute

  • face
  • size
  • color
  • bold

attribute-value

face size color bold
gfont 000% 000000 yes
mfont no

explanation

fontタグは明朝体、ゴシック体を指定したい句や文に使用します。

example

<p><font face="mfont" size="130%" bold="yes" color="ff0000">私が明朝体です。</font></p>
<p><font face="gfont" size="130%" bold="no" color="ff0000">私がゴシック体です。</font></p>
