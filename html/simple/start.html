start

説明

startタグは、字下げに使用します。

property
  • タグ名称
  • start
  • 属性
  • val
  • 属性値
  • Xem
  • XemXX
  • クラス変形
  • <div class="start-XemXX">

CSSプロパティ

.start-XemXX
margin-top: X.XXem
