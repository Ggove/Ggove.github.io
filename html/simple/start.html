start

説明

startタグは、字下げに使用します。

property

タグ名称

start

属性

val

属性値

  • [1-9]*[0-9]+em[0-9]*

クラス変形

(.vrtl | .hltr) .start-XemYY

スタイルシートプロパティ

  • margin-(left | top): X.YYem

example

<start val="1em125">これは長い文章です。これは長い長い文章です。これは長い長い長い文章です。長くて退屈な文章です。内容だけは空っぽです。</start>
