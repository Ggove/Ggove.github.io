●img

tag name

  • img

attribute

  • src
  • alt
  • class

attribute-value

src alt class
画像へのpath 代替文字 スタイル指定

explanation

imgタグは画像を挿入します。inline要素のため、画像と文章の行を分けたい場合には、block要素(ex. div,p)で囲む必要があります。

example

<start val="1em"><h1 class="gfont font-1em30">第一章</h1></start>
<p><br /></p>
<p>　画像を挿入します。<img src="image/sample.jpg" alt="pumpkin"/>　画像を挿入しました。</p>
