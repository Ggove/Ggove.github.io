indent

説明

indentタグは、行頭の字下げに使用します。

property
  • タグ名称
  • indent
  • 属性
  • val
  • 属性値
  • Xem
  • XemXX
  • クラス変形
  • class="indent-XemXX"
  • CSSプロパティ
  • text-indent: X.XXem

※現在小数点以下はコンバートエラーのため使用できません。

example

Code

<indent val="2em">いろはにほへと、ちりぬるを、わかよたれそ、つねならむ。うゐのおくやま、けふこえて、あさきゆめみし、ゑひもせす。</indent>
