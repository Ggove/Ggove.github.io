hindent

説明

hindentタグは、文章が折り返されたときの折り返しインデントを指定します。

property
  • タグ名称
  • hindent
  • 属性
  • val
  • 属性値
  • Xem
  • XemXX
  • クラス変形
  • <div class="h-indent-XemXX">

CSSプロパティ

text-indent padding-left ( hltr ) padding-top ( vrtl )
-X.XXem X.XXem X.XXem
example

Code

<hindent val="3em">これは長い文章です。これは長い長い文章です。これは長い長い長い文章です。長くて退屈な文章です。内容だけは空っぽです。</hindent>

illustration

図1: hindent = 3

Point

indentタグと組み合わせて使用することが可能です。例えば次のように書きます。


<hindent val="3em"><indent val="1em">これは長い文章です。これは長い長い文章です。これは長い長い長い文章です。長くて退屈な文章です。内容だけは空っぽです。</indent></hindent>


また、属性値には、任意の自然数を指定できます。 例えば、
<hindent val="12em125">
等、

Caution

hindentタグとindentタグの組み合わせで段落を表現することは非推奨です。代わりにstartタグを使用することが推奨されています。


hindentタグとindentタグを逆順にして使用することはできません。プロパティの書き換え順で最終的な値が変更されてしまうからです。


例えば、下のように書いたとします。


<indent val="1em"><hindent val="3em">これは長い文章です。これは長い長い文章です。これは長い長い長い文章です。長くて退屈な文章です。内容だけは空っぽです。</hindent></indent>


これは、text-indentプロパティをhindentが上書きしたことによる現象です。これによってindentタグの効力がなくなっています。
