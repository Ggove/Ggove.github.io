gaiji

説明

gaijiタグは、文章中に変換できない文字や文字列がある場合に、代替の手段として画像を使用するために使います。

property
  • タグ名称
  • gaiji
  • 属性
  • src
  • alt
  • class

attribute-value

src alt class
文字列 代替文字 スタイル指定 gaiji gaiji-line gaiji-wide
  • タグ変形
  • gaiji → img

クラス変形

なし

CSSプロパティ

gaiji gaiji-line gaiji-wide
width="1em" width="1em" width=auto
width="1em" height=auto height="1em"
その他 その他 その他
example

Code

<p>　王<gaiji src="image/u83BE.png" alt="莾"/><!--莾-->は紀元８年に国号を「新」と改めた。王<gaiji class="gaiji" src="image/u83BE.png" alt="莾"/><!--莾-->は武帝が官学とした儒教を狂信しており、儒教に則った政治を徹底しようとした。</p>
