bookinfo

説明

bookinfoタグは、論理目次を生成します。その際、タグで囲んだテキストを本文と論理目次に記述します。

property

タグ名称

bookinfo

属性

なし

属性値

なし

xml変形

metadata

スタイルシートプロパティ

なし

example

Code

<bookinfo>
<seriesname></seriesname>
<serieskana></serieskana>
<maintitlename>佐藤さんの休日は最低だ</maintitlename>
<maintitlekana>サトウサンノキュウジツハサイテイダ</maintitlekana>
<subtitlename>なし</subtitlename>
<subtitlekana>なし</subtitlekana>
<creatorname num="001">Y.J.</creatorname> <
creatorkana num="001">ワイジェー</creatorkana>
<creatorrole num="001">aut</creatorrole>
<publishername>(有)Yznet</publishername>
<publisherkana>ユウゲンガイシャワイズネット</ publisherkana>
<epubspec>editor</epubspec>
<language>ja</language>
<uuid>自動</uuid>
<isbn>なし</isbn>
<update>自動</update>
<ebpajver>1.1.3</ebpajver>
<ibooksver>1.0</ibooksver>
<ibooksscroll>自動</ibooksscroll>
<bookshelf>image/cover.jpg</bookshelf>
<linesize>1.6</linesize>
<pgdirection>rtl</pgdirection>
<linkline>あり</linkline>
<addncx>なし</addncx>
<vhtoc>自動</vhtoc>
</bookinfo>

Point

ほとんどのオプションが決まっているため、コピペで対応可。
ただし、maintitlename, maintitlekana, creatorname, creatorkana, publishername, publisherkanaは毎回変わるので入力すること。
